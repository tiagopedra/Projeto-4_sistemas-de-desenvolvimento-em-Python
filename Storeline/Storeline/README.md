🛍️ Estoque e Compras - Loja de Roupas

###
📋 Descrição

Este projeto consiste em um sistema simples desenvolvido
em Python para gerenciamento de uma loja de roupas. 
O sistema permite cadastrar produtos,controlar o estoque e 
realizar vendas, atualizando automaticamente as quantidades disponíveis.

###
⚙️ Funcionalidades
👕 Cadastro de roupas (nome, código, quantidade e valor)
📦 Visualização do estoque
💰 Sistema de vendas
📉 Atualização automática do estoque após vendas
⚠️ Aviso de estoque baixo
📊 Relatório com:
    Quantidade de peças em estoque
    Quantidade de peças vendidas
    Total vendido (faturamento)

###
🏗️ Estrutura do Projeto
LojaDeRoupas/
│
├── main.py        # Menu principal do sistema
├── vendas.py      # Sistema de vendas
├── estoque.py     # Controle de estoque
└── roupa.py       # Classe Roupa

###
🧠 Conceitos Utilizados
Programação Orientada a Objetos (POO)
Classes e objetos
Métodos construtores (__init__)
Encapsulamento
Listas para armazenamento de dados
Tratamento de erros (try/except)

###
🚀 Como Executar
Certifique-se de ter o Python instalado (3.14)
Execute o arquivo principal:
python main.py
Utilize o menu para navegar pelo sistema.

###
📊 Exemplo de Uso
Cadastrar uma roupa
Visualizar estoque
Realizar uma venda
Ver relatório com dados atualizados

###
⚠️ Observações
O valor apresentado como "lucro" representa o total vendido (faturamento), pois o sistema não considera o custo das peças.
Os dados são armazenados apenas durante a execução (não há banco de dados).
